# Continuous Assignment with Feedback

## Problem

Design a combinational circuit using continuous assignments.

Logic implemented:
```
B = A AND C (with 3 ns delay)
C = NOT B (with 2 ns delay)
```
It creates a feedback loop between B and C.

---

## Design File

**conti_assign.v**

Implement the logic using **continous assignment (`assign`)** statements.

---

## Testbench

**tb_conti_assign.v**

The testbench performs the following tasks:

- Generates input stimulus for A
- Changes A from 0 to 1 at time 5 ns
- Displays simulation results using "$monitor"
- Ends simulation using "$finish"

## Note:
B and C are generated by the design and are not driven by the testbench.
